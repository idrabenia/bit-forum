КЛАСС: User

ПРИМЕР: 
    // получаем объект юзера
    $auth = User::getInstance();
    
    // получаем никнейм пользователя
    $login = $auth->getUserNickname();
    
    // получаем id пользователя
    $id = $auth->getUserId();
    
    // проверяем права пользователя
    if ($auth->isAdmin()) {
        echo 'Это Одмин';
    }
    else if ($auth->isGuest()) {
        echo 'Это гость';
    }
    else if ($auth->isRegisteredUser()) {
        echo 'Это обычный юзер';
    }
    
    // залогиниваем пользователя. в функцию передаются
    // логин, пароль и флаг, говорящий о необходимости запомнить 
    // пользователя
    $auth->login('Логин', 'Пароль', TRUE);
    
    // Разлогиниваем пользователя
    $auth->logout();
    
    // Есть константы типа int, соответствующие правам пользователя
    // в базе данных
    echo User::GUEST;
    echo User::ADMINISTRATOR;
    echo User::REGISTERED_USER;
    
    
МЕТОДЫ: 
    1) ИМЯ: getInstance()
       ОПИСАНИЕ: Метод возвращает экземпляр объекта User.
       В скрипте может быть только один пользователь. Поэтому 
       был применён паттерн Singleton, и экземпляр возвращается 
       через статический метод класса, что гарантирует, что в программе
       будет использоваться только один объект User.
       
    2) ИМЯ: getUserNickname()
       ОПИСАНИЕ: Для зарегистрированных пользователей метод возвращает
       логин текущего пользователя, иначе метод возвращает FALSE.
       
    3) ИМЯ: getUserId()
       ОПИСАНИЕ: Метод возвращает id, если пользователь зарегистрирован
       и FALSE иначе.
       
    4) ИМЯ: isAdmin()
       ОПИСАНИЕ: Метод возвращает TRUE если пользователь имеет права админа,
       иначе метод возвращает FASLE.
       
    5) ИМЯ: isGuest()
       ОПИСАНИЕ: Метод возвращает TRUE если пользователь является гостем,
       иначе метод возвращает FASLE.
       
    6) ИМЯ: isRegisteredUser()
       ОПИСАНИЕ: Метод возвращает TRUE если пользователь имеет права зарегистрированного
       пользователя форума, иначе метод возвращает FASLE.
       
    7) ИМЯ: login($login, $password, $remember = FALSE)
       ОПИСАНИЕ: Метод осуществляет аутентификацию пользователя.
       
    8) ИМЯ: logout()
       ОПИСАНИЕ: Метод осуществляет разлогинивание пользователя.
